Flickr Photo for JeyKyll
=======================

Description:
------------
convert Flickr Photo URL to Photo Title & Image Link

Files:
------

    .
    └── plugins/
      └── flickr-photo.rb

Syntax:
-------
    {% flickr PHOTO_ID SIZE %}
| arg | detail |
|---|---|
| PHOTO_ID  | flickr photo id |
| SIZE | size label resized by flickr|
